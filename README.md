# speech-emotion
detect emotion in an audio clip

Classify speech as: 
```
emotions={
  '01':'neutral',
  '02':'calm',
  '03':'happy',
  '04':'sad',
  '05':'angry',
  '06':'fearful',
  '07':'disgust',
  '08':'surprised'
}
```

Using the librosa library for analyzing audio. 

Trained on Ryerson Audio-Visual Database of Emotional Speech and Song dataset
